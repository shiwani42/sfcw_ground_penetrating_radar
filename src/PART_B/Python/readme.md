# Python Script for Data Acquisition

This folder contains a Python script: **`sample.py`**

---

## 📄 Description

The script reads data transmitted from the **STM32 microcontroller** via serial communication and saves it to a file named **`sample.txt`**. This file is later used for signal processing in MATLAB.

---

## 🛠️ Customization

- The **path to `sample.txt`** can be modified within the script to match the path used in the MATLAB code for seamless integration.

---

This script plays a crucial role in bridging the hardware data acquisition with the signal processing pipeline.
